Cat Litterbox Liner Calculator

Catlitterboxlinercalculator.com is a website I created from scratch using HTML, CSS and Javascript. I published it using Google Domains and Github Pages. The purpose is for a user to measure their litterbox and input their findings so that the calculator will designate a properly sized cat pan liner for the user to purchase.

NFL Spread Generator

NFL Spread Generator is an application that creates betting spreads based on an algorithm and user-generated data. The purpose of this program is to compare the spreads to the popular Vegas' spreads and look for any discrepancies. If the two disagree on a spread, the objective was for the Spread Generator to be correct. If so, any further discrepancies were betting targets. Additionally, the user is allowed to update team statistics on a weekly basis based on real life score results, which causes the application to adjust its spreads accordingly.

Alfapet Pallet Builder is an application that was created to better organize the layout of pallet orders on the warehouse floor. Since the warehouse has limited space, it is recommended to get the most out of it. There is also additional criteria that a particular pallet order must meet, such as:

• A pallet order must not be over 6 layers tall in height
• A pallet order must not have more than 4 pallets per pallet order
• A pallet order must be limited to at most one 40 case order per pallet order

The so-called "laying out" of pallet orders was a laborous task performed by hand but now it is performed by a program that does it in a matter of seconds.
